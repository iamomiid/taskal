# Code of Conduct

## Our Pledge

We want participation in this project to be an experience that is respectful, constructive, and welcoming for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:

- being respectful and considerate in discussions
- offering actionable, good-faith feedback
- assuming good intent while still being clear about problems
- helping make the project easier to understand and use

Examples of unacceptable behavior:

- harassment, discrimination, or personal attacks
- trolling, intimidation, or sustained disruption
- publishing private information without permission
- bad-faith misrepresentation of another person's work or intent

## Scope

This Code of Conduct applies in project spaces and in public spaces when someone is representing the project.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that do not align with this Code of Conduct.

## Reporting

To report unacceptable behavior, please contact the project maintainer privately through GitHub.
